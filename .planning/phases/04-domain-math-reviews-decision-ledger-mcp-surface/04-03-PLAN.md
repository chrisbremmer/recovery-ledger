- `median(values: number[]): number` ÔÇö standard (n+1)/2-th value
- `medianAbsoluteDeviation(values: number[]): number` ÔÇö raw MAD, NOT scaled
- `wilcoxonRankSum(sampleX: number[], sampleY: number[]): number` ÔÇö rank sum for sampleX only (NOT U, NOT p)
- `cumulativeStdNormalProbability(z: number): number` ÔÇö Phi(z) standard-normal CDF

Conversions (RESEARCH ┬žStatistical Engine):
- U_1 = R_1 Ôłĺ n_1┬Ě(n_1 + 1) / 2  (where R_1 = wilcoxonRankSum return)
- mu_U = n_1 ┬Ě n_2 / 2
- sigma_U = sqrt(n_1 ┬Ě n_2 ┬Ě (n_1 + n_2 + 1) / 12)
- z = (|U Ôłĺ mu_U| Ôłĺ 0.5) / sigma_U  (continuity correction)
- p_two = 2 ┬Ě (1 Ôłĺ Phi(z))

REV-01 explicit requirement: MAD scaled by 1.4826 (Rousseeuw & Croux 1993).

    3. `mannWhitney(sampleX: number[], sampleY: number[]): { U: number; p: number }` ÔÇö wraps `wilcoxonRankSum` + computes U + computes two-sided p via normal approximation with continuity correction.
